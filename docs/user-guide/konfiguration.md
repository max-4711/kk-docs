# Konfiguration

## Projekt erstellen/öffnen

Nach dem Programmstart muss zunächst für die entsprechende Veranstaltung ein neues Projekt erstellt oder ein bereits existierendes geöffnet werden. Dies geht über die Menüzeile am oberen Rand des Fensters: "Datei" > "Neu"/"Öffnen". In beiden Fällen muss ein Ordner angegeben werden, in dem die zugehörigen Dateien abgelegt werden, bzw. aus dem sie gelesen werden.
