# Begrifflichkeiten

Diese Seite gibt Überblick über diverse Begrifflichkeiten, die im Zusammenhang mit der Verwendung der Software immer wieder auftauchen.

**Projekt**: Ein Ordner, der alle Daten einer Veranstaltung enthält: Ablauf, Einstellungen, Infos der Kandidatinnen, verwendete Grafiken, usw. Zu erkennen an der darin enthaltenen `project.json`-Datei.
