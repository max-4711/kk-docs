# Begrifflichkeiten (Glossar)

Diese Seite gibt Überblick über diverse Begrifflichkeiten, die im Zusammenhang mit der Verwendung der Software immer wieder auftauchen.

**Projekt**: Ein Ordner, der alle Daten einer Veranstaltung enthält: Ablauf, Einstellungen, Infos der Kandidatinnen, verwendete Grafiken, usw. Zu erkennen an der darin enthaltenen `project.json`-Datei.

**Seite**: Eine Seite ist ein Eintrag im konfigurierten Veranstaltungsablauf, bzw. etwas, was auf dem Bildschirm angezeigt wird. Gewissermaßen ähnlich wie eine Folie bei PowerPoint. Jede Seite hat eigene Einstellungen (welche genau, hängt vom jeweiligen Typ ab), die _nicht_ zwischen verschiedenen Seiten gleichen Typs synchronisiert werden, d.h. jede Seite hat ihre ganz eigenen Einstellungen.

**Seitentyp**: Jede Seite hat einen Typ, bzw. genau genommen können andersrum vom jeweiligen Typ dann Seiten erstellt werden. Die Liste aller verfügbaren Seitentypen sind in der Administrationsansicht unter "Neue Seite via Drag'n'Drop zum Wahlablauf hinzufügen" zu sehen.
 
**Kapitel**: Eine Gruppe von Seiten im Veranstaltungsablauf - dient aber lediglich der Übersichtlichkeit. Zudem können in der aktuellen Programmversion keine neuen Kapitel manuell erstellt werden.

**Projekteinstellungen**: Diese Einstellungen gelten für das gesamte Projekt, also nicht nur für einzelne Seiten. Das sind die Einstellungen, die unter "Projekt" > "Einstellungen" konfiguriert werden.

**Seiteneinstellungen**: Diese Einstellungen gelten nur für jeweils eine Seite. Das sind die Einstellungen, die in der Administrationsansicht unter "Gewählte Seite des Wahlablaufs konfigurieren" getätigt werden können.

**Programmeinstellungen**: Diese Einstellungen werden nicht im Projekt gespeichert, sondern sind unabhängig davon - sie beziehen sich auf den Computer, auf dem sie getätigt werden. Das betrifft insbesondere die Einstellungen der Lautstärkemessung (Menü "Plugins" > "Lautstärkemessung" > "Einstellungen"). Achtung: Wird das Projekt von einem Computer auf einen anderen übertragen, müssen die Programmeinstellungen dort von neuem getätigt werden!

**Bilder-Pool**: Die meisten Bilder eines Projekts sind in verschiedenen Pools organisiert. Ein Pool versteht sich dabei ähnlich wie ein Ordner bzw. ein Fotoalbum. Der Benutzer kann beliebig viele Bilder-Pools erstellen und jeweils beliebig viele Bilder hinzufügen (Menü "Projekt" > "Bilder-Pools" > "Neuen Bilder-Pool erstellen" / "Neues Bild hinzufügen"). In der aktuellen Programmversion gibt es nur einen einzigen Typ von Bilder-Pool (bei dem Dateien von der Festplatte des Computers ausgewählt werden können), in vorherigen Versionen existierten auch andere Typen, um z.B. Bilder von Instagram zu einem wählbaren Hashtag anzeigen zu können.

**Administrationsansicht**: Das ist das Programmfenster, welches der Bediener der Software hauptsächlich sieht, bzw. in dem er die Software bedienen kann.

**Präsentationsansicht**: Das ist die Ansicht, welche die Zuschauer der Veranstaltung sehen.

**Ressource**: Dabei handelt es sich um Dateien, also fast immer Bilder, die in das Projekt geladen werden.

**Kalibrierung**: Prozess, um die Mikrofone zu testen, um die Genauigkeit der Audiomessung zu erhöhen.
