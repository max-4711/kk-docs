# Test (Steuerung des Programms im Wahlmodus)

Nachdem der Veranstaltungsablauf konfiguriert ist, kann das Programm über den Button "Wahl starten" (am unteren Bildschirmrand) in den Wahlmodus versetzt werden.

Der Grundsätzliche Aufbau in der Administrationsansicht bleibt erhalten, dennoch gibt es einen wichtigen Unterschied:

![Administrationsansicht: Überblick Wahlmodus](images/040-wahlmodus.png)

An der Stelle der Liste/Auswahl von neuen Seiten befindet sich nun die Navigation, die das Wechseln zur nächsten (oder Rückkehr zur vorherigen) Seite im Veranstaltungsablauf erlaubt.

Zusätzlich ist es immer auch möglich, über einen Rechtsklick auf eine Seite im Wahlablauf (also auf der linken Bildschirmseite) und "Diese Seite präsentieren" direkt zur jeweiligen Seite zu springen.

Der Bereich für die Einstellungen bzw. die Interaktion mit der aktuellen Seite zeigt, grundsätzlich die Optionen der für das Publikum gerade sichtbaren Seite. Aber Achtung: Wenn eine Seite links im Wahlablauf angeklickt wird, werden die Optionen dieser Seite angezeigt (auch wenn sie gerade für das Publikum nicht zu sehen ist!). Wenn sich die für das Publikum sichtbare Seite ändert (z.B. via Klick auf Vor/Zurück), springt die Anzeige jedoch wieder zu den Einstellungen/der Steuerung der dann sichtbaren Seite.

Während der Wahlmodus aktiv ist, sind einige Einstellungen nicht mehr verfügbar. Insbesondere die Liste der Kandidatinnen ist dann eingefroren und kann nicht mehr verändert werden, solange der Wahlmodus aktiv ist.

Dafür werden zwei neue Optionen in der Menüleiste verfügbar: Über "Projekt" > "Punkte zurücksetzen" können sämtliche bislang in Spielen erfasste Punktzahlen der Kandidatinnen wieder gelöscht werden. Im regulären Veranstaltungsablauf wird diese Funktion nicht benötigt, kann während der Testphase aber nützlich sein.

Sowohl während im Testablauf als auch während der Veranstaltung nützlich kann hingegen die über "Projekt" > "Zwischenstand" erreichbare Funktion sein, die ein neues Fenster mit der aktuellen Ergebnistabelle (unsichtbar für das Publikum) öffnet.

> ℹ️ **Speicherung von Punkten**
>
> Zwischenstände bzw. Punkte bleiben vom Programm gespeichert, auch wenn die Wahl beendet und erneut gestartet wird (solange das Programm selbst geöffnet bleibt).
> Beim Beenden des Programms hingegen werden die Punkte gelöscht.
