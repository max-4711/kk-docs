# Kirschenkrönung

Dieses Repository, bzw. diese Webseite enthält die Dokumentation und Teile des Quellcodes von "Kirschenkrönung".

"Kirschenkrönung" ist die Software, welche die Wahl der [Kirschenkönigin](https://www.gladenbach.de/kultur-tourismus/kirschenmarkt/kirschenkoeniginnen/) auf dem [Gladenbacher Kirschenmarkt](https://www.gladenbach.de/kultur-tourismus/kirschenmarkt/) medial begleitet.
Erstmals zum Einsatz kam die Software nach damals knapp einjähriger Entwicklungszeit auf dem Kirschenmarkt 2018. 

Der vollständige Quellcode der Software befindet sich im Repository [max-4711/kirschenkroenung](https://github.com/max-4711/kirschenkroenung). Leider ist dieses Repository derzeit aber nicht öffentlich verfügbar - der Grund ist juristischer Natur. Im Repository enthalten sind neben dem Code selbst auch diverse Grafiken/Icons etc. die nicht selbst erstellt, sondern lizenziert wurden. Und die Lizenzen gestatten lediglich die Veröffentlichung als Teil eines Gesamtwerks (ergo: der Software), nicht aber der entsprechenden Artefakte selbst.

Auf formlose Anfrage (entweder über die [Stadt Gladenbach](https://www.gladenbach.de/) oder die Kontaktoptionen des GitHub-Profils [max-4711](https://github.com/max-4711)) kann jedoch gerne Zugriff auf den Quellcode gewährt werden.
